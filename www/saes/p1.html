Packet Tracer

Packet Tracer is Cisco's network hardware simulation software
It allows you to design virtual network architectures, so that :

  • Students learn to network without physical hardware
  • Professionals can test certain architectures before actually implementing them.
  • All this is done exclusively with Cisco virtual equipment.

    Routing project

    This part of the project taught us how to operate an infrastructure with several networks.
    All the networks had to communicate with each other by configuring routers.
    To make it all work, I had to :

  • Configure routes so that networks can communicate with each other
  • Configure router interfaces with the correct IP addresses
  • Configuring the switch management interface

    • After testing, all devices ping each other.

    Vlan switching

    This part of the project taught us how to do Inter-Vlan routing.


    For this I had to :
  • Configure router subinterfaces with the correct IP addresses
  • Configure switch interfaces with the correct IP addresses
  • Set up VLANs on switches
  • Inter-Vlan routing configuration
  • Enabling the right Vlans in trunk mode

    • Once routing is complete, all vlans can communicate.

    Skills acquired

    I was able to develop several skills such as :

  • Switch and router configuration: Ability to configure switches and routers using specific commands
  • Spanning-Tree principle: Understanding the concepts of Spanning-Tree Protocol (STP) and its variants to avoid loops in networks.
  • Inter-Vlan Routing: Ability to configure routing between different virtual networks (VLANs) to enable communication between them.
  • Trunk configuration: Ability to configure trunks, which allow multiple VLANs to be transported over a single physical connection.
