Design and set up a complete infrastructure



In this project, we were in groups of 5 students. Our aim was to design and implement a functional infrastructure for a company, based on ANSSI recommendations. At the end of the allotted time, an AUDIT will be carried out by another group.
For my part, I had to set up a secure Active Directory infrastructure.

Project goal

In this project, in a team of 5, we designed an enterprise infrastructure from scratch, including the creation of a demilitarized zone (DMZ) and the configuration of an internal information system. The only instruction given was to scrupulously follow ANSSI (National Agency for Information Systems Security) recommendations throughout the process.

Key components :


  • Servers x2 (ESXI 6.7.0)
  • Switch (Dell Power Connect 5324)
  • Router G7 (Cisco 1941 Series)
  • A WEB server (Apache)
  • Physical firewall x2 (WatchGuard M270 12.8.2)
  • Software firewall (StormShield 4.2.13)
  • A virtual machine dedicated to Active Directory
  • PRTG
  • DNS
