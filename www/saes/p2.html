Pentest

It aims to identify potential vulnerabilities in a system, network or application, using attack techniques.

Main techniques used during a pentest :

  • Scanning and recognition: gathering information on the target.
  • Network penetration testing: assessment of network infrastructure weaknesses.
  • White-box penetration testing: evaluation of a system with detailed information on the infrastructure.
  • Black-box penetration testing: evaluation of a system without any information about the infrastructure.
